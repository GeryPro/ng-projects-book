# IssueTracker

This project an updated version of Chapter 03's project from the book "Angular Projects - Second Edition" from Aristeidis Bampakos. The main difference is that one form component is used both creating and editing an issue instead of the original two separate components.

It was generated with [Angular CLI](https://github.com/angular/angular-cli) version 12.1.3.

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.
